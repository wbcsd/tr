## Website Documents

This folder contains all downloadable resources listed on the PACT website: https://www.carbon-transparency.org/resources

Each document here is publicly available under the following site:

```
https://wbcsd.github.io/tr/websitedocs/
```

## Publishing a document

- First, give the document a clear non-confusing name, optionally include a version or a date.
- Make sure that the name does not contain spaces or special characters, replace these with '-' dashes.
- Click [Add File] in the top-right corner
- Upload the file, and save (=commit) it, enter a short description.

- Within a few minutes, the document will be downloadable via

  ```https://wbcsd.github.io/tr/websitedocs/<document-name>```
